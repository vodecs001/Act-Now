# 快办 / Act Now

Act Now は、日々のタスクをすぐ確認できる Windows 向けの軽量なデスクトップ付箋 Todo アプリです。画面端に置いて、追加、完了、編集、アーカイブをすばやく行えます。

## 現在のリリース

現在はポータブル版のみ提供しています。インストーラー版はありません。

```text
release\快办-免安装.exe
```

実行ファイルをダブルクリックすると起動できます。旧バージョンが起動中の場合は、先に終了してから新しいポータブル版を開いてください。

## 機能

- Todo：追加、完了、編集、削除に対応し、完了したタスクは自動でアーカイブされます。
- Todo 編集：各 Todo の右側にある鉛筆アイコン、または Todo テキストのダブルクリックで編集できます。
- 作成時間：各 Todo の左下に作成時間を表示します。
- グループ：すべての Todo とカスタムグループを利用できます。グループを削除しても、その中の Todo は「すべて」に残ります。
- 旧デフォルトグループの整理：古いデータに残った `默认 / inbox` グループを自動で移行し、削除できないデフォルトグループが左側に表示されないようにします。
- アーカイブ：完了済み Todo の復元、個別削除、全消去に対応します。
- 画面端に隠す：左、右、上の画面端に隠せます。下端では自動非表示になりません。
- 上端自動収納：ウィンドウを画面上端へドラッグすると隠れ、マウスを近づけると表示されます。
- ウィンドウ操作：ドラッグ移動、ドラッグリサイズ、タスクバーへの最小化、閉じるボタンの動作設定に対応します。
- 自動起動：Windows 起動時に自動起動できます。
- 多言語：中文、English、日本語に対応し、メイン画面とトレイメニューも切り替わります。
- Material You 風：明るい配色、柔らかいコンテナ色、見やすい入力欄、タイトルバーのホバー表示を備えています。
- タスクバーとトレイ：起動中は Windows のタスクバーとシステムトレイに表示されます。

## ソースから実行

Windows と Python 3 が必要です。

```powershell
python sticky_todo.py
```

`启动便签.bat` を実行することもできます。

## データ保存場所

ユーザーデータは以下に保存されます。

```text
%APPDATA%\ActNow\data.json
```

プロジェクトフォルダーをコピーまたは置き換えても、このユーザーデータは自動では消えません。現在のバージョンは起動時に古いデフォルトグループを自動移行します。

## ビルド

現在のリリースではポータブル実行ファイルのみを残します。

```text
release\快办-免安装.exe
```

このプロジェクトは PyInstaller でパッケージ化します。

```powershell
scripts\build_release.ps1
```
