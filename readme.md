# 快办 / Act Now

快办是一款轻量的 Windows 桌面便签待办工具，适合把当天要处理的事情放在桌面边缘，随手记录、勾选、编辑和归档。

## 当前版本

当前只提供免安装版，不再提供安装程序。

```text
release\快办-免安装.exe
```

双击即可运行。运行前如果旧版还开着，请先退出旧版，再打开新的免安装版。

## 功能

- 待办事项：支持添加、勾选完成、编辑、删除；完成后自动进入归档。
- 待办编辑：每条待办右侧提供铅笔图标按钮，也可以双击待办文字进行编辑。
- 创建时间：每条待办左下角显示创建时间。
- 分组管理：支持“所有待办”和自建分组；删除分组时，组内待办会保留在“所有待办”中。
- 默认分组清理：会自动清理旧数据中残留的“默认 / inbox”分组，避免左侧出现无法删除的默认分组。
- 归档箱：保存所有已完成待办，支持恢复、单条删除和清空归档。
- 贴边隐藏：支持左侧、右侧、顶部贴边隐藏，底部不会触发贴边。
- 顶部自动收起：拖到屏幕顶部后自动收起，鼠标靠近后弹出。
- 窗口控制：支持拖拽移动、拖拽缩放、最小化到任务栏、关闭时退出或最小化到托盘。
- 开机自启动：可在设置里打开或关闭。
- 多语言：支持中文、英文、日文，主界面和托盘右键菜单会随语言切换。
- Material You 风格：浅色界面、柔和容器色、清晰输入框和顶部按钮悬停高亮。
- 托盘与任务栏：运行时显示在任务栏和系统托盘。

## 源码运行

需要 Windows 和 Python 3。进入项目目录后运行：

```powershell
python sticky_todo.py
```

也可以双击 `启动便签.bat`。

## 数据位置

软件数据保存在当前 Windows 用户目录：

```text
%APPDATA%\ActNow\data.json
```

复制或替换项目文件不会自动清空这份用户数据。当前版本会在启动时自动迁移旧默认分组。

## 打包

当前发布只保留免安装版：

```text
release\快办-免安装.exe
```

项目使用 PyInstaller 打包，可参考：

```powershell
scripts\build_release.ps1
```
